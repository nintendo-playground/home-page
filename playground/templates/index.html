Welcome!

Messing around with Nintendo's servers is risky because it may lead to a device ban.

This website lets you play around with various servers without contacting Nintendo's servers at all. The servers implement the functionality of the original servers as accurately as possible, but there may be some differences. Currently, the servers support everything up to system version 14.1.2.

The source code of these services can be found here.

Note: because I regenerated the CAs the playground currently does not accept any certificates. New certificates can be downloaded in the resources section below soon.
